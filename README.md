# 📊 Log Analyzer

## 🚀 About the Project

This project is a simple log analyzer that helps in understanding system logs and detecting unusual activities. It combines basic rule-based checks with machine learning and NLP techniques to find patterns, errors, and anomalies in the logs.

It also includes USB monitoring to track device usage, making it useful for basic security analysis.

---

## ✨ What it does

* Finds errors and warnings using rule-based logic
* Detects unusual patterns using machine learning
* Uses NLP (TF-IDF and BERT) to analyze log data
* Tracks USB device activity
* Shows results through a simple Streamlit interface

---

## 🛠️ Technologies Used

* Python
* Machine Learning
* NLP (TF-IDF, BERT)
* Streamlit
